Как запускать?

Запуск сервера
```
docker-compose up -d --build
```

Simple chat (без стриминга)
```
python3 simple_console_chat.py localhost:8080
```

Console chat (со стримингом)
```
python3 console_chat.py --server localhost:8080
```
